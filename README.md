## Webpack5 + TypeScript 4 实现贪吃蛇

## 特性

- [x] Webpack5
- [x] TypeScript 4
- [x] Babel
- [x] less+postcss
- [x] webpack-dev-server
- [x] Jest
- [x] Eslint

## 怎么运行？

# 安装依赖.

$ npm i

# 启动服务器:

npm start

# 打包一个版本，对输出不压缩:

npm run build

# 打包一个版本，对输出压缩处理:

npm run build.prod

# 运行单元测试:

npm run test
